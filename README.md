# ZombieInvadersVR
A Virtual Reality Game using Google Cardboard and libGDX.
This game is basically a final work to my graduate studies (Brazil Pós Graduação).

# System Requirements
* Android Smartphone/Cellphone
* Gyroscope Sensor (in your Smartphone)
